# star2026
A folder to organize all of our files throughout the 2026 STAR program

1) initialCA: these are all of the initial files and scripts for the CA, copied over from the bls directory mentioned in the implementation directions pdf. They contain the configuration files for the CA, implementation of the CA in go, and the driver programs to run the CA in python 2.

2) python3Translations: these are all of the driver programs from the initial CA files translated into python 3, along with all of the tools we needed for the translation. They can be run in your tux terminal.
