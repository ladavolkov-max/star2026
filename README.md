# star2026
A folder to organize all of our files throughout the 2026 STAR program

1) initialCA: all of the initial files and scripts for the CA, copied over from the bls directory mentioned in the implementation directions pdf. They contain the configuration files for the CA, implementation of the CA in go, and the driver programs to run the CA in python 2.

2) python3Translations: all of the driver programs from the initial CA files translated into python 3, along with all of the tools we needed for the translation. They can be run in your tux terminal.

3) filesForDesktop: the files related to running the CA on your local computer. These include the CA itself, along with the different configurations that can be used to structure its behavior. These should be kept on your desktop because our current code that uses them tells the coputer to look for them specifically on the desktop.
